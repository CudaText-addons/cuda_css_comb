CSS Comb plugin for CudaText.
It uses CSScomb engine to sort CSS (also SCSS, LESS) props in specific order. 

You can configure it in json file.
Doc about config https://github.com/csscomb/csscomb.js/blob/master/doc/options.md

**** You need to install Node.js first from official site.
